AlexIzydorczyk/sudoku
Ce projet est un jeu de Sudoku en C++. L’utilisateur peut choisir une grille prédéfinie, remplir les cases manuellement et demander de l’aide : le programme remplira automatiquement une case correcte.
L'équipe est : Alexandre Berlia, Théophile Roulland Durrande, Gildas Montcel.
